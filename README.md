# Practice11 - Pygame practice games.
## This project contains mini-games built using Python and PyGame, extended from Practice 10 with additional features and improvements.
It is organized into three main modules:
1.**Racer game**
Added improvements:
1)Randomly generated coins with different weights (values)
2)Enemy speed increases after collecting N coins
3)Code is fully commented for better understanding
2.**Snake game**
Added improvements:
1)Randomly generated food with different weights (points)
2)Food disappears after a certain time (timer-based)
3)Code is fully commented
3.**Paint application**
Added improvements:
1)Draw square
2)Draw right triangle
3)Draw equilateral triangle
4)Draw rhombus
